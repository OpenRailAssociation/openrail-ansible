# OpenRail Ansible Playbooks and Roles

![OpenRail Administrative Project](https://openrailassociation.org/badges/openrail-project-admin.svg)

This repository contains Ansible playbooks and roles for deploying and managing systems owned by OpenRail Association. The playbooks are designed to automate the setup and maintenance of various services and infrastructure components.
